# Prácticas de Nuevas Tecnologías y Empresa - UBU

En este repositorio subo mis 3 prácticas que hice en Nuevas Tecnologías y Empresa el año 2025-2026 de la UBU con mi compañero Roberto Iturralde Zarzo. 

## 📂 Contenido del Repositorio
Además de subir las prácticas, también subo las instrucciones y el PDF Informativo sobre cómo se valoran las prácticas. Y por último subo mis notas de toda la asignatura en un excel.

La estructura del proyecto es la siguiente:
* 📁 **PRACTICAS/**: Contiene una subcarpeta dedicada a cada una de las 3 prácticas.
* 📄 **Notas_Asignatura.xlsx**: Registro de las calificaciones.
* 📄 **PDF_Informativo.pdf**: Criterios de evaluación de las prácticas.

## ⚙️ Requisitos para la Ejecución

**Para las prácticas de Python (Vacunas y Redes Aleatorias):**
* Tener **Anaconda Navigator** descargado e instalado.
* Tener **Jupyter Notebook 7.2.2** (o superior).
* Tener Python instalado nativamente y Visual Studio 2022.
* Al abrir el Jupyter le podremos dar a Ejecutar y se ejecutará el programa en Python 3.12.4.final.0.

**Para la práctica de Análisis Visual:**
* Descargar e instalar **Gephi** desde su [página oficial](https://gephi.org/).
* Es requisito indispensable tener **Java** instalado en el equipo para poder abrir el proyecto.
